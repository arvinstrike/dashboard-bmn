Custom Alert System

Professional & Modern Alert System untuk Dashboard BMN

Version 1.0

Fitur Utama

  • Animasi smooth dan modern
  • 4 tipe alert: Success, Error, Warning, Info
  • Toast notifications
  • Confirmation dialogs
  • Responsive design
  • Backward compatible dengan SweetAlert2 API
  • Tidak memerlukan library eksternal

Modal Alerts

Alert modal yang muncul di tengah layar dengan backdrop blur. Cocok untuk notifikasi penting.

Contoh Kode:
Swal.fire({
  title: 'Berhasil!',
  text: 'Data berhasil disimpan',
  icon: 'success'
});

Toast Notifications

Notifikasi kecil yang muncul di pojok kanan atas. Otomatis hilang setelah beberapa detik. Cocok untuk feedback cepat.

Contoh Kode:
CustomAlert.toast({
  title: 'Berhasil!',
  message: 'Data berhasil disimpan',
  icon: 'success',
  duration: 3000
});

Confirmation Dialogs

Dialog konfirmasi untuk aksi penting seperti delete atau submit. Memiliki dua tombol: Confirm dan Cancel.

Contoh Kode:
confirmDelete({
  itemName: 'data pengajuan BMN',
  onConfirm: function() {
    // Kode untuk delete data
  }
});

Helper Functions

Shorthand functions untuk kemudahan penggunaan. Tidak perlu menulis object config lengkap.

Contoh Kode:
showSuccess('Data berhasil disimpan');
showError('Gagal menyimpan data');
successToast('Data tersimpan');
errorToast('Terjadi kesalahan');
